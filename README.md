그래도 강사님이 열정으로 설명해주시니 8비트는 8핀이있어야되는거고 16비트는 16핀 등등
1은전기가 흐르고 0은 전기가안흐르고 그래서 핀마다 흐르고안흐르고 규칙을 정해서 CPU가 만들어진다는걸
처음 알았다. 프로그래밍언어로 컴파일 방식하는 구조는 보면 이해가 되는데 설명이 안돼서 더 복습하고 공부해야겟다.
